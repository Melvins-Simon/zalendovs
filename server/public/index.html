Zalendo Voting System API Documentation

API endpoints for managing election system

Authentication

POST /signup
Register a new user account. Requires username, password, and role.
POST /login
Authenticate a user and return an access token. Requires username and password.

Representatives Data

GET /facultyreps
Retrieve a list of all faculty representatives.
GET /classreps
Retrieve a list of all class representatives.
GET /delegates
Retrieve a list of all delegates.

Voting System

POST /vote
Submit a vote for a candidate. Requires candidate ID and voter details.
GET /results
Retrieve current election results.
POST /reset-results
Reset all election results (admin only).

Reviews

POST /review
Submit a review/feedback about the election process.
GET /reviews
Retrieve all submitted reviews.
